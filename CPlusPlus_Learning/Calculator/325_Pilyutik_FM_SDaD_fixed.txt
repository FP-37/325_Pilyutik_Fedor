Тест №1:
(2,5),(2,7),(3,10)
#
true
#
В данном треугольнике сумма любых двух сторон > третьей стороны
=> треугольник существует.


Тест №2:
(4,0),(8,0),(10,0)
#
false
#
Зная, что координаты по оси ординат одинаковы, мы можем осознать, что треугольник уже не мог бы получится,
а функция подтвердит это, так как сумма двух меньших сторон будет равна большей,
что соответствует условию возвращения значения false и математически оправдывает невозможность такого треугольника.


Тест №3:
(5,5),(5,5),(5,5)
#
false
#
Случай совпадения точек => мы понимаем, что треугольник не может существовать.
Программа подтвердит отсутствие треугольника, так как при любом варианте
сумма двух сторон треугольника даёт в результате версию 0+0=0.

Тест №4:
(4,2),(-2,-4),(6,-6)
#
true
#
Случай равностороннего треугольника => треугольник однозначно существует.
Программа подтвердит существование треугольника, так как любые две стороны будут в два раза больше третьей.


Тест №5:
(0,0),(0,8),(0,6)
#
true
#
Случай прямоугольного "Египетского" треугольника - катеты равны 6 и 8, гипотенуза равна 10.
Программа проверит три случая:
6+10 <= 8 - false НЕ сработает,
8+10 <= 6 - false НЕ сработает,
6+8 <= 10 false НЕ сработает.
