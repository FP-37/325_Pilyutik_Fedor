# Playfair Cipher

## Описание
Реализация шифра Плейфера (5x5 таблица, I=J). Поддерживает шифрование и дешифрование английских текстов.

## Возможности
- Объединение I и J
- Поддержка повторяющихся букв
- Автоматическое добавление X в пары

## Требования
- Python 3.6+
